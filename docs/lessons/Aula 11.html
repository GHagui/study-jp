🧠 Nível 2 — Complete

  1. "Este morango está gostoso" → このいちごは ＿＿＿ です。
  2. "A água está gelada" → みずは ＿＿＿ です。
  3. "Eu como maçã" → りんごを ＿＿＿。
  4. Diferença entre さわやか e つめたい?
Ver Respostas
  1. このいちごは おいしい です (oishii)
  2. みずは つめたい です (tsumetai — gelado ao toque)
  3. りんごを たべます (tabemasu — como)
  4. さわやか = refrescante (sensação). つめたい = gelado (temperatura física). São sensações diferentes!
