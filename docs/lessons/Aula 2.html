🎌 Aula 2 — Verbos de Sala de Aula e Pronomes

🎯 O que você vai aprender

Ao final desta aula, você será capaz de:

  • Dar comandos educados em sala de aula usando 〜てください
  • Entender o padrão de conjugação て (te-form) dos verbos
  • Usar pronomes pessoais corretos para cada gênero e situação
  • Conduzir uma conversa simples de sala de aula em japonês

📚 Verbos e Comandos com ください (Kudasai)

📄 Estrutura

[Verbo na forma -te] + ください (kudasai) = Por favor, [verbo]!

ください sozinho = "por favor" (dá/me dê)

🇯🇵 ください (kudasai) vem do verbo くださる (kudasaru) = "conceder/dar (a mim)" — é um verbo honorífico. Você literalmente diz "conceda-me [fazer tal ação]".

🇯🇵 Dicionário Romaji 🇧🇷 Significado Forma て Comando Completo Significado
よむ yomu ler よんで よんでください yonde kudasai Por favor, leia
きく kiku ouvir きいて きいてください kiite kudasai Por favor, ouça
かく kaku escrever かいて かいてください kaite kudasai Por favor, escreva
みる miru ver / olhar みて みてください mite kudasai Por favor, olhe/veja
いう iu dizer いって いってください itte kudasai Por favor, diga
たつ tatsu levantar たって たってください tatte kudasai Por favor, levante-se
すわる suwaru sentar すわって すわってください suwatte kudasai Por favor, sente-se

🧩 O Segredo da Forma て (te-form)

A forma て é essencial no japonês — você vai usá-la para comandos, pedidos, conectar frases, e muito mais. O segredo está na última sílaba do verbo na forma dicionário:

〜く → 〜いて

かく → かいて (kaku → kaite)

きく → きいて (kiku → kiite)

🧠 Mnemônica: "K de Kaká vai de ITEms"
〜む / 〜ぶ / 〜ぬ → 〜んで

よむ → よんで (yomu → yonde)

あそぶ → あそんで (asobu → asonde)

🧠 Mnemônica: "MoBo NDa — M, B, N viram NDe"
〜る / 〜う / 〜つ → 〜って

すわる → すわって (suwaru → suwatte)

いう → いって (iu → itte)

たつ → たって (tatsu → tatte)

🧠 Mnemônica: "RUTsu vira TTE — pense em RUTa TTe"
〜す → 〜して

はなす → はなして (hanasu → hanashite)

🧠 Mnemônica: "SU de SUshi vai pra SHIte"
〜ぐ → 〜いで

およぐ → およいで (oyogu → oyoide)

(não aparece nesta aula, mas grave o padrão!)

🧠 Mnemônica: "GU de água — vira IDe (molhado → で com tenten)"

⚠️ Exceção Importante: いく (iku)

O verbo いく (iku) = "ir" segue a regra く→いて normalmente, mas merece atenção especial porque é muito frequente:

いく → いって (iku → itte)

💡 Não confunda: いって (itte) pode significar "vá" (de いく) OU "diga" (de いう). O contexto decide!

💎 Níveis de Polidez

De mais direto → mais educado

FormaNívelQuando usar
よんで yonde Casual Entre amigos, familiares. Pode soar rude com desconhecidos.
よんでください yonde kudasai Educado Sala de aula, colegas, situações cotidianas. USE ESTE.
よんでくださいませんか yonde kudasaimasen ka Muito educado "Você não faria o favor de ler?" — Com superiores, clientes.

💡 Regra prática

Na sala de aula japonesa, o professor usa 〜てください com os alunos. É educado sem ser distante. Alunos respondem ao professor com linguagem ainda mais polida (けいご / keigo).

Entre colegas de classe, pode-se usar a forma casual (sem ください), especialmente em grupos de estudo informais.

👤 Pronomes Pessoais

ℹ️ Contexto Cultural

Em japonês, o gênero do falante e a situação social determinam qual pronome usar. Diferente do português, onde só existe "eu" e "você", o japonês tem múltiplas formas com nuances de gênero, idade, e hierarquia.

🇯🇵 Hiragana Romaji 🇧🇷 Significado Gênero Situação
わたしwatashieu Neutro Formal / Padrão. Use em qualquer situação.
わたくしwatakushieu Neutro Muito formal. Discursos, negócios, cerimônias.
ぼくbokueu Masculino Casual/íntimo. Homens jovens, entre amigos.
おれoreeu Masculino Bem casual. Entre amigos homens. Não use com superiores!
あたしatashieu Feminino Casual. Mulheres jovens, entre amigas.
あなたanatavocê Neutro Padrão. Evite usar com superiores (use o nome!).
きみkimivocê Neutro Casual/íntimo. Com amigos próximos ou namorado(a).

⚠️ Regra de Ouro

Na dúvida, use わたし (watashi). É seguro em 100% das situações.

Evite あなた (anata) com superiores. Em vez disso, use o nome da pessoa + さん (san):

❌ あなたは日本人ですか？ (anata wa nihonjin desu ka?)
✅ 田中さんは日本人ですか？ (Tanaka-san wa nihonjin desu ka?)

Na prática, japoneses omitem o sujeito sempre que o contexto já está claro — esta é uma das maiores diferenças do português!

🎯 Exercícios

📝 Nível 1 — Traduza para Japonês

  1. Por favor, leia. → ?
  2. Por favor, escreva. → ?
  3. Por favor, ouça. → ?
  4. Por favor, sente-se. → ?
  5. Por favor, diga mais uma vez. → ?
Ver Respostas
  1. よんでください (yonde kudasai)
  2. かいてください (kaite kudasai)
  3. きいてください (kiite kudasai)
  4. すわってください (suwatte kudasai)
  5. もういちど いってください (mou ichido itte kudasai)

🧠 Nível 2 — Qual pronome usar?

  1. Você é homem, falando com seu chefe. → ?
  2. Você é mulher, falando com amigas. → ?
  3. Você está numa entrevista de emprego. → ?
  4. Você é homem, falando com amigos próximos (bem casual). → ?
  5. Você quer chamar a atenção de um colega (não use あなた!). → ?
Ver Respostas
  1. わたし (watashi) — formal com superiores. Nunca おれ!
  2. あたし (atashi) — casual feminino entre amigas
  3. わたし (watashi) ou わたくし (watakushi) — máxima formalidade
  4. おれ (ore) — bem casual masculino, ou ぼく (boku) se for mais suave
  5. Nome + さん! Ex: 田中さん (Tanaka-san). Evite あなた.

🧩 Nível 3 — Conjugue para forma て

  1. かく (escrever) → ?
  2. よむ (ler) → ?
  3. いう (dizer) → ?
  4. たつ (levantar) → ?
  5. きく (ouvir) → ?
Ver Respostas
  1. かいて (kaite) — く→いて
  2. よんで (yonde) — む→んで
  3. いって (itte) — う→って
  4. たって (tatte) — つ→って
