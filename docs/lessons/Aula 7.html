Kata x Hira nos Lugares
Palavras em カタカナ são empréstimos estrangeiros:
ホテル (hotel), レストラン (restaurant), コンビニ (convenience store), バー (bar)

Palavras em ひらがな/漢字 são japonesas originais:
ぎんこう, としょかん, がっこう, えき

🏪 Padrão 〜や (ya) = Loja de ~

Muitas palavras para lojas terminam com や (ya) = "loja/estabelecimento":

  • ほん (hon) = livro → ほんや (honya) = livraria (loja de livros)
  • はな (hana) = flor → はなや (hanaya) = floricultura (loja de flores)
  • パン (pan) = pão → パンや (panya) = padaria (loja de pão)

Regra: [produto] + や = loja que vende [produto]! 🧠

💬 Pedindo e Dando Direções

Diálogo Conversação Direções

Estrutura do Diálogo

  1. Chamar atenção: すみません (Sumimasen) = "Com licença"
  2. Perguntar localização: [lugar]はどこですか？ = "Onde fica [lugar]?"
  3. Responder com posição: [lugar A]は[lugar B]の[posição]です。 = "[Lugar A] fica [posição] de [Lugar B]."
  4. Agradecer: ありがとうございます (Arigatogozaimasu) = "Obrigado/a"
  5. Responder: どういたしまして (Doitashimashite) = "De nada / Imagina"

📝 Diálogo 1 — Procurando o Banco

A: すみません。
Sumimasen.
→ Com licença.

A: ぎんこうはどこですか？
Ginkō wa doko desuka?
→ Onde fica o banco?

B: ぎんこうはこうえんのちかくです。
Ginkō wa kōen no chikaku desu.
→ O banco fica perto do parque.

A: ありがとうございます。
Arigatogozaimasu.
→ Obrigado/a!

B: どういたしまして。
Doitashimashite.
→ De nada!

📝 Diálogo 2 — Procurando o Hotel

A: すみません。ホテルはどこですか？
Sumimasen. Hoteru wa doko desuka?
→ Com licença. Onde fica o hotel?

B: ホテルはレストランのうしろです。
Hoteru wa resutoran no ushiro desu.
→ O hotel fica atrás do restaurante.

A: ありがとうございます。
Arigatogozaimasu.
→ Obrigado/a!

B: どういたしまして。
Doitashimashite.
→ De nada!

🎯 Mini-Quiz

Teste Quiz Desafio

📝 Nível 1 — Traduza

  1. Banco → ?
  2. Biblioteca → ?
  3. Em cima → ?
  4. Direita → ?
  5. Norte → ?
Ver Respostas
  1. ぎんこう (ginkou)
  2. としょかん (toshokan)
  3. うえ (ue)
  4. みぎ (migi)
  5. きた (kita)

🧠 Nível 2 — Complete

  1. "Onde fica o cinema?" → えいがかんは ＿＿＿ ですか？
  2. "O hotel fica atrás do restaurante" → ホテルはレストランの ＿＿＿ です。
  3. Loja de pão → パン + や = ＿＿＿
  4. "Com licença" → ＿＿＿
Ver Respostas
  1. えいがかんは どこ ですか？ (doko = onde)
  2. ホテルはレストランの うしろ です (ushiro = atrás)
  3. パンや (panya) — padrão [produto] + や!
  4. すみません (sumimasen)

🗺️ Nível 3 — Dê Direções

Usando これ/それ/あれ (isto/isso/aquilo — básico), indique:

  1. "O parque fica perto da estação" → ?
  2. "A cafeteria fica na frente da escola" → ?
  3. "A farmácia fica à esquerda do supermercado" → ?
Ver Respostas
  1. こうえんは えきの ちかく です (kouen wa eki no chikaku desu)
  2. きっさてんは がっこうの まえ です (kissaten wa gakkou no mae desu)
  3. やっきょくは スーパーの ひだり です (yakkyoku wa suupaa no hidari desu)
