Como Importar no Anki

Guia passo a passo para importar os baralhos no Anki Desktop e AnkiDroid.

1

Abra o Anki

Inicie o Anki Desktop no seu computador ou o AnkiDroid no celular.

💡 Se ainda não tem, baixe em apps.ankiweb.net (gratuito).
2

Clique em "Importar Ficheiro"

No menu principal, clique em Importar Ficheiro (ou Import File se estiver em inglês).

Selecione o arquivo .txt que você baixou.

3

Configure o separador de campos

Na janela de importação, escolha o separador correto:

  • Basic Bunpo → Separador: Ponto e vírgula
  • Flashcards N5 Melhorado → Separador: Tab
⚠️ Confira a pré-visualização. As colunas devem aparecer alinhadas corretamente.
4

Mapeie os campos

Associe as colunas do arquivo aos campos do tipo de nota:

  • Basic Bunpo: Campo 1 → Frente, Campo 2 → Verso
  • Flashcards N5 Melhorado: Campo 1 → Frente, Campo 2 → Verso

Tipo de nota sugerido: Basic (ou Básico).

5

Escolha o baralho de destino

Selecione um baralho existente ou crie um novo:

  • Basic Bunpo → Crie: Japonês::Vocabulário Básico
  • Flashcards N5 Melhorado → Crie: Japonês::Gramática N5
💡 Use :: (dois pontos duplos) para criar sub-baralhos organizados.
6

Confirme a importação

Clique em Importar. O Anki mostrará quantos cartões foram adicionados.

Pronto! Os cartões já estão disponíveis para estudo.

🎉 Dica: Configure revisões diárias em Opções do Baralho para manter a consistência.
